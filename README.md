# The Convivio Site

This project is built using Jekyll and Gulp.

## Dependencies

[NVM](https://github.com/creationix/nvm), [RVM](https://rvm.io/), and [Bundler](http://bundler.io/)

## Installing

In the project root folder, run:
1. `nvm use`
1. `npm install --global gulp-cli`
1. `nvm use`
1. `rvm use`
1. `bundle install`
